### Protocol Event Quick Reference

| Protocol | Transport | Content Type | Use Cases |
|----------|-----------|--------------|-----------|
| **AG-UI** | SSE | `text/event-stream` | Agent streaming, state updates, tool calls |
| **A2UI** | JSONL | `application/x-ndjson` | Generative UI rendering |
| **DCRL** | SSE + REST | `application/json` | Confidence-based conversational building |
| **A2A** | HTTP/JSON-RPC | `application/json` | Cross-builder agent coordination |

## Usage Instructions for Stitch

1. **Open Google Stitch** (stitch.withgoogle.com)
2. **Start New Project** with "Hyyve Platform"
3. **Apply Design System** from Group 1 global tokens
4. **Generate screens in order** by group
5. **Copy each prompt** and paste into Stitch
6. **Review and iterate** on generated output
7. **Export to React** when satisfied
8. **Import to AI Studio** for Gemini integration
