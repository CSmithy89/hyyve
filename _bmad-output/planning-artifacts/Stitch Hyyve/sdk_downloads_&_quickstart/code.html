Quickstart Guide

Follow these steps to integrate Hyyve in under 5 minutes.

1

Initialize the Client

Import the library and configure it with your API key from the dashboard.

import hyyve

# Initialize with your API Key
client = hyyve.Client(
    api_key="hv_live_...",
    environment="production"
)
2

Define your Workflow

Create a workflow object defining the parameters and expected output.

workflow = client.workflows.create(
    name="data-processing-v1",
    timeout=300, # seconds
    retries=3
)
3

Execute and Handle Results

Run the workflow asynchronously and await the results.

async def main():
    result = await workflow.execute({ "input_id": 12345 })
    print(f"Workflow finished: {result.status}")

# Run the async loop
hyyve.run(main())
