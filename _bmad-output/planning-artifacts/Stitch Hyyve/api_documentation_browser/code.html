POST

/v1/execute

Trigger a new workflow execution asynchronously. This endpoint accepts a payload to initialize the workflow state and returns an execution ID for tracking.

lock

Authorizations

Requires a valid API token with write:workflows scope.

Body Parameters

Param Type Description
workflow_id REQUIRED string The unique identifier (UUID) of the workflow definition to execute.
inputs REQUIRED object A JSON object containing key-value pairs that map to the workflow's input variables.
priority enum Execution priority. Can be low, normal, or high. Defaults to normal.
callback_url string An optional webhook URL that will be called with the result payload once execution completes.

Response Codes

202 Accepted. Execution has been queued.
400 Bad Request. Missing required parameters or invalid JSON.
401 Unauthorized. Invalid API token. 
const axios = require('axios');

const options = {
  method: 'POST',
  url: 'https://api.hyyve.io/v1/execute',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_TOKEN'
  },
  data: {
    workflow_id: 'wf_89234-abcd-55',
    inputs: {
      email: 'user@example.com',
      plan: 'enterprise'
    }
  }
};

axios.request(options).then(function (response) {
  console.log(response.data);
}).catch(function (error) {
  console.error(error);
});
