read_file

MCP Tool

Reads a file from the filesystem. This tool allows the agent to access local data for processing.

integration_instructions Tool Documentation & Schema
expand_more

Description

Use this tool to read the contents of a file at the specified path. The user (or the preceding node) must provide an absolute path string. The tool returns the full text content of the file.
